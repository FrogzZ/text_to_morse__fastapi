<h2>Переводчик азбуки Морзе. Языки: RU, ENG</h2>
<p><b>Установка:</b></p>
<ul>
  <li>pip install fastapi</li>
  <li>pip install "uvicorn[standard]"</li>
  <li>pip install Jinja2</li>
</ul>

<p><b>Запуск:</b></p>
<ul>
  <li>uvicorn main:app --reload</li>
</ul>
